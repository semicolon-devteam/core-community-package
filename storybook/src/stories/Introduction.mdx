## 🎯 개요

**@team-semicolon/community-core**는 세미콜론 커뮤니티 플랫폼의 핵심 기능을 제공하는 라이브러리입니다.
개발자들이 빠르고 일관성 있게 커뮤니티 기능을 구현할 수 있도록 Services, Hooks, Utils를 제공합니다.

## ✨ 주요 기능

### 🪝 Hooks
- **인증 시스템** (`useAuth`)
- **전역 상태 관리** (`useGlobalLoader`)
- **React Query** 통합 데이터 페칭
- **반응형 유틸리티** (`useDeviceType`)

### 🔧 Services
- **REST API** 클라이언트
- **인증 및 권한 관리**
- **파일 업로드 및 관리**
- **에러 핸들링 시스템**

### 🛠️ Utilities
- **날짜/시간 처리**
- **숫자 포맷팅**
- **이미지 최적화**
- **JWT 토큰 관리**

## 📋 빠른 시작

### 1. 패키지 초기화

```tsx
import { initializeCommunityCore } from '@team-semicolon/community-core';

initializeCommunityCore({
  apiUrl: process.env.REACT_APP_API_URL,
  supabase: {
    url: process.env.REACT_APP_SUPABASE_URL,
    anonKey: process.env.REACT_APP_SUPABASE_ANON_KEY,
  }
});
```

### 2. 훅 사용


## 📚 문서 구조

### Hooks
- **Authentication**: 인증 관련 훅들
- **Data Fetching**: React Query 기반 데이터 페칭 훅들
- **Utilities**: 유틸리티 훅들

### Services
- **API Services**: REST API 클라이언트들
- **Authentication**: 인증 및 권한 관리
- **File Management**: 파일 업로드 및 관리

## 🎨 디자인 시스템

이 라이브러리는 일관성 있는 사용자 경험을 위해 통합된 디자인 시스템을 제공합니다:

- **색상 팔레트**: Primary, Secondary, Success, Warning, Error
- **타이포그래피**: Pretendard 폰트 기반 한국어 최적화
- **간격 시스템**: 8px 그리드 시스템
- **그림자와 테두리**: 계층감 있는 UI 요소들

## 🌟 주요 특징

- **🔧 개발자 친화적**: TypeScript 완전 지원으로 뛰어난 DX 제공
